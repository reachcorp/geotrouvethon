# geotrouvethon

## Installation des librairies

    pip install -r requirements.txt
    
## Usage
Launch the server

    geotrouvethon.py
    
Call API with

    http://127.0.0.1:9966/locate/<place>
    
***ex:***

- http://127.0.0.1:9966/locate/honolulu\
==> *[21.3250785,-157.846644178573]*

- http://127.0.0.1:9966/locate/zzzzzzzz\
 ==> *[-99,99]*
